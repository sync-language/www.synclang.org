Sync is an easily embeddable programming language for building robust, scalable software with parallel performance and memory safety.

Embeddable

Sync is lightweight, being embeddable easily into applications using any build system. A simple API is provided, with native support for C, C++, Rust, and Zig. All Sync structs use the C memory layout, and no garbage collector is used, allowing you to pass data to and from Sync without any translation layers. Sync can also be built on all platforms that have a standard C++17 or higher compiler.

Multithreading

Sync supports running all built code in parallel, without data races and deadlocks. It is well-suited to use all of the processing power available in a system.

Safety

Thread safety is memory safety. Sync's ownership model guarantees modern safety at compile-time while also being able to communicate safely with the program it is embedded in.

Flexibility

Sync code is compiled when you want while your host program is running. This allows maximum flexibility in what your program can do.
